# Shoppin_category_classification
## 프로젝트 목적
타이틀만으로 어떤 카테고리의 제품인지 분류해주는 모델을 만들고자 했습니다.<br/>
ex) 폴햄 스프링 페스타 티셔츠 -> 패션으로 분류, 삼성 갤럭시북4 가성비노트북 대학생 -> 디지털/가전으로 분류 <br/>
